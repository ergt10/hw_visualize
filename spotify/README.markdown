### 数据来源

根据 **spotify-top-200-dataset.csv** ，选取了 2017/1/6 榜单的数据制作了 Spotify Top 200 音频特征径向可视化柱状图。

### 图表信息

排名前200的歌曲从12点方向开始顺时针排列，从舞曲性、能量、调性、调式、拍号、整体响度、语词率、原声度、纯音乐度、现场感、欢快度、节拍率这12个音乐特征的维度出发，用线段长度表示值的高低，并计算了平均值。

可以选择不同的分析维度；在图标上查看对应的信息；鼠标在数轴上滑动时，图表会高亮显示数值最接近的歌曲信息。

### 打开方式

在终端cd到对应文件夹，输入 `python3 -m http.server 8000`。

连接成功后新建终端窗口，输入 `open http://localhost:8000/spotify.html`，即可打开网页。
